Matryx PasswordGen is a simple tool to generate secure passwords quickly.
Whether you need multiple passwords or just one strong password, this tool is perfect for you.


Key Features:

Generate Multiple Passwords: Customizable quantity and length.
Enhanced Security: Passwords include lowercase letters, numbers, and uppercase letters.
Colorful Interface: A pleasant command-line interface using colorama.
Customization: Choose the length (minimum 3 characters).


How It Works:

Enter the number of passwords.
Select the desired length for each.
The generator creates secure passwords.
Passwords are displayed one by one.


Matryx PasswordGen - v0.1 is ideal for anyone needing secure and unique passwords.
